How DeBiasMe Works

Our Approach

DeBiasMe employs a comprehensive approach to bias detection and visualization, using AI to analyze text for potential biases while providing interactive visualizations to help users understand and address these biases.

The Process

1. Bias Detection

When you enter text, DeBiasMe uses Azure OpenAI to analyze it against an extensive taxonomy of bias types. The system identifies potentially biased phrases and categorizes them according to a hierarchical classification system.

2. Visualization

Detected biases are visualized using interactive D3.js force-directed graphs that show relationships between different types of biases found in the text. This hierarchical visualization helps users understand the broader categories and specific manifestations of bias.

3. Highlighting & Suggestions

Biased phrases are highlighted directly in the text, with tooltips explaining the type of bias detected. Each highlighted section includes suggestions for alternative, more neutral phrasing.

4. Interactive Learning

Users can click on highlighted phrases or nodes in the visualization to learn more about specific bias types. The system provides educational content about why certain language patterns may perpetuate bias.

5. Refinement

Users can accept suggestions to automatically replace biased phrases with more neutral alternatives, helping to develop more inclusive communication habits over time.

Technical Implementation

DeBiasMe is built with modern web technologies:

  • Pure JavaScript with ES6 modules for a lightweight, efficient experience
  • D3.js for interactive data visualizations
  • Azure OpenAI API for sophisticated bias detection
  • Custom state management for a responsive user interface
  • Mobile-responsive design for accessibility across devices
